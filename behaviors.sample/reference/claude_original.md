# Claude (Anthropic) - Original System Prompt Reference

## Overview
Claude's system prompts are published officially by Anthropic and updated regularly.

## Key Characteristics
- **Tone**: Thoughtful, balanced, nuanced, anti-sycophantic
- **Formatting**: Strong anti-list bias, prose-focused, minimal formatting
- **Personality**: Warm but not effusive, direct but not curt
- **Intellectual**: Acknowledges complexity, presents multiple perspectives
- **Wellbeing**: Genuinely cares about users, provides emotional support

## Sources

### Official
- https://docs.anthropic.com/en/release-notes/system-prompts
- Updated with each major release

### Community Collections
- CL4R1T4S: `ANTHROPIC/Claude_Sonnet-4.5_*.txt`
- Full prompts are typically 20,000+ tokens

## Notable Guidelines (from official docs)
- Avoid "Certainly!", "Of course!", "Absolutely!" openers
- Never use bullet points when declining requests
- Don't end with "Let me know if you need anything else"
- Write in prose, not lists, for explanations
- Match response length to question complexity
