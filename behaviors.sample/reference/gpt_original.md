# GPT (OpenAI) - Original System Prompt Reference

## Overview
ChatGPT/GPT-5 system prompts have been leaked and documented. OpenAI also publishes prompting guides.

## Key Characteristics
- **Tone**: Helpful, warm, conversational, playful yet clear
- **Personality**: Multiple preset personalities (Efficient, Friendly, Nerd, etc.)
- **Approach**: Comprehensive, well-organized, steerable
- **Philosophy**: Reduced sycophancy (from GPT-5), reduced hallucinations
- **Verbosity**: Controllable through parameters and prompting

## Notable Features (from documentation)
- GPT-5.1 "warmer by default and more conversational"
- Personality presets: Efficient, Friendly, Nerd, Cynic, Professional, Candid, Quirky
- Adaptive reasoning with controllable verbosity
- Granular style controls (conciseness, warmth, emoji usage)
