# Grok (xAI) - Original System Prompt Reference

## Overview
Grok is designed with a distinct personality modeled after The Hitchhiker's Guide to the Galaxy - knowledgeable but irreverent.

## Key Characteristics
- **Tone**: Witty, sarcastic, bold, unfiltered
- **Personality**: Rebellious streak, tackles controversial topics
- **Approach**: Direct, honest, minimal filtering
- **Modes**: Regular mode and "Fun Mode" (more unhinged)
- **Philosophy**: Anti-"woke", less censored than competitors

## Sources

### Community Collections
- CL4R1T4S: `XAI/GROK-4.1_*.txt`
- Multiple versions from Grok 3 to Grok 4.1

## Notable Features (from research)
- Inspired by Douglas Adams' Hitchhiker's Guide
- "Incredibly knowledgeable but also incredibly sarcastic"
- Fun Mode: "more likely to be sarcastic, use slang, and give answers that are fun"
- Willingness to address controversial topics
- Instruction tuning shapes the rebellious personality

## Considerations
Grok's unfiltered nature can sometimes lead to inaccuracies or biased information. The adapted versions maintain the direct, witty personality while being more measured.
