# DeepSeek - Original System Prompt Reference

## Overview
DeepSeek is known for analytical reasoning, chain-of-thought processes, and steady, methodical responses.

## Key Characteristics
- **Style**: Analytical, steady, focused, calming
- **Reasoning**: Chain-of-thought, self-verification, reflection
- **Personality**: Patient, supportive, seeks harmony
- **Approach**: Methodical, takes time to read carefully
- **Values**: Structure, consistency, thoroughness

## Sources

### Community Collections
- Various leaked prompts in community repositories
- DeepSeek-R1 documentation: https://huggingface.co/deepseek-ai/DeepSeek-R1

## Notable Features (from research)
- "Reasoning model" comparable to OpenAI o1
- Self-verification and reflection capabilities
- Generates long chains of thought (CoTs)
- DISC profile: Steadiness personality (SIc or SCi)
- Results tend to be more specific due to careful reading

## Technical Capabilities
- DeepSeek-R1 trained via reinforcement learning
- Excels in logical inference and mathematical reasoning
- Scored 91.6% on DROP (reasoning questions) - highest among tested models
